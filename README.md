# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to business insights visualization. The project involves data cleaning, exploratory data analysis (EDA), SQL analysis using PostgreSQL, interactive dashboard creation in Power BI, and documentation through a detailed report and presentation.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.


## Dataset

The dataset contains business-related records used for analysis.

**Key Information**

* Structured tabular dataset
* Multiple numerical and categorical features
* Used for data cleaning, analysis, SQL querying, and dashboard creation

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Examined data structure and data types.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data.
* Converted data types where necessary.

### 3. Exploratory Data Analysis (EDA)

* Summary statistics
* Distribution analysis
* Correlation analysis
* Trend identification
* Data visualization

### 4. SQL Analysis (PostgreSQL)

* Imported cleaned dataset into PostgreSQL.
* Wrote SQL queries to answer business questions.
* Used:

  * SELECT
  * WHERE
  * GROUP BY
  * ORDER BY
  * JOIN
  * Aggregate Functions
  * CASE Statements

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* KPIs
* Sales/Performance Trends
* Category Analysis
* Regional Analysis
* Interactive Filters & Slicers
* Charts and Visualizations

### 6. Report & Presentation

* Prepared a detailed project report.
* Created a presentation using Gamma summarizing:

  * Problem Statement
  * Analysis Process
  * Key Findings
  * Business Recommendations

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook files.
4. Run the notebooks in sequence:

   * Data Loading
   * Data Cleaning
   * EDA
5. Import the cleaned dataset into PostgreSQL.
6. Execute the SQL queries.
7. Open the Power BI (`.pbix`) file to explore the dashboard.
8. Review the project report and presentation for detailed findings.
